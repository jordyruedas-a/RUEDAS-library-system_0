# RUEDAS-library-system_0

A lightweight, early-stage prototype of a library management system built as part of an Object-Oriented Programming course. 

This version demonstrates the fundamental logic of book registration, user handling, and sales tracking. It serves as the foundational base for the more advanced, full-stack RUEDAS Library System.

**Tech Stack:** HTML, CSS, Python (Flask), MongoDB.
**Status:** Academic prototype / Legacy version.
